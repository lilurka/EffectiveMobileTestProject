## 🔧 Как устроено

┌─────────────┐
│ Браузер │
└──────┬──────┘
│ HTTP (порт 80)
↓
┌─────────────┐
│ Nginx │ ← Раздаёт фронтенд (HTML)
│ Reverse │ Проксирует API запросы
│ Proxy │
└──────┬──────┘
│ Внутренняя сеть Docker
↓
┌─────────────┐
│ Backend │ ← Недоступен извне
│ :8080 │ Только через Nginx
└─────────────┘


- **Nginx** — открыт наружу, раздаёт фронтенд и проксирует API
- **Backend** — не имеет прямого доступа из интернета
- **Вся инфраструктура** — в Docker Compose

## 🐳 Стек

- Docker + Docker Compose
- Nginx (reverse proxy)
- Python (http.server)
- Yandex Cloud VPS
