## 🔧 Как устроено

| Компонент | Роль | Доступность |
|-----------|------|-------------|
| **Браузер** | Пользовательский интерфейс | Внешний |
| ↓ | | |
| **Nginx (80)** | Reverse proxy + фронтенд | Открыт наружу |
| ↓ | | |
| **Backend (8080)** | API сервер | Только внутри Docker |


- **Nginx** — открыт наружу, раздаёт фронтенд и проксирует API
- **Backend** — не имеет прямого доступа из интернета
- **Вся инфраструктура** — в Docker Compose

## 🐳 Стек

- Docker + Docker Compose
- Nginx (reverse proxy)
- Python (http.server)
- Yandex Cloud VPS
